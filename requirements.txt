# 🐉 小龙人PSPAI 依赖清单
# 安装：pip install -r requirements.txt
# ── 必须（引擎启动） ──
# 小龙人PSPAI纯标准库即可启动，无需任何pip包
#
# ── 可选（工具功能增强） ──
# excel_read     需要: openpyxl
# pdf_extract    需要: pymupdf
# ssh_exec       需要: paramiko
# ai_search      需要: requests
# config.yaml    需要: pyyaml（已支持config.json纯标准库回退）
#
# 安装全部可选工具：
#   pip install openpyxl pymupdf paramiko requests pyyaml
#
# 只安装你需要的：
#   pip install requests          # 搜索功能
#   pip install openpyxl          # Excel读取
#   pip install pymupdf           # PDF提取
#   pip install paramiko          # SSH远程执行
