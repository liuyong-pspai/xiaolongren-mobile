🐉 小龙人·数字生命体 — 使用说明
══════════════════════════════════

📦 首次使用
1. 解压本文件夹到任意位置（推荐桌面）
2. 双击「安装桌面快捷方式.bat」— 桌面出现图标
3. 双击桌面「小龙人·数字生命体」图标

⚙ 配置API Key
浏览器打开后 → 点右上角⚙ → 输入你的API Key → 保存
选一个角色，开始聊天！

▶ 以后每次
双击桌面「小龙人·数字生命体」图标即可

📌 需要 Python 3.10+（https://www.python.org/downloads/）
   安装时记得勾选 "Add Python to PATH"

支持模型：DeepSeek / OpenAI / Anthropic / 通义千问 / 智谱
